SmartCalcGtkC

Описание

Программа представляет собой умный калькулятор, имеющий два режима работы.

Первый режим реализует инженерную версию калькулятора, принимающего на вход сложные скобочные выражения с возможностью подстановки переменной с заданным значением. Имеет простой интерфейс с окном ввода и кнопками (цифры, операции, функции и т.д.).

На вход программы могут подаваться как целые, так и вещественные числа, записанные через точку.

В поле для ввода выражения может быть введено произвольное скобочное арифметическое выражение в инфиксной нотации в том числе с подстановкой значения переменной x.

Значение переменной x вводится в отдельном поле с помощью клавиатуры.

Указание оператора умножения * перед переменной x после числа является необязательным.

По умолчанию значение переменной x установлено в ноль.

Для удаления последнего символа в поле ввода используйте DEL, а для очистки всех полей ввода используйте AC.

Во втором режиме производится построение графика в соответствии с выражением, введённым в поле ввода выражения во вкладке Graph.

Реализация

Реализованы следующие арифметические операции и математические функции:

Скобки (), сложение +, вычитание -, умножение *, деление /, остаток от деления mod, возведение в степень ^, унарный плюс +, унарный минус -, тригонометрические функции (sin, cos, tan, asin, acos, atan), вычисление квадратного корня sqrt, вычисление десятичного логарифма log, вычисление натурального логарифма ln.

Программа разработана на языке Си стандарта C11 с использованием компилятора gcc и библиотеки элементов интерфейса GTK4.

Программа работает по алгоритму Дейкстры, каждое выражение переводится в обратную польскую запись, а затем вычисляется.
